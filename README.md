# Market Watch · A 股异动市场体检

每日自动生成的 A 股市场异动数据看板。

## 数据来源

数据由 [`trend-trading-agents`](https://github.com/kangjinghang/trend-trading-agents) 的 scan-all 流程采集（雪球全市场异动快照），经脱敏聚合后 push 到本仓库。

## 内容

- **异动密度**：当日入选候选股占全市场比例 + 历史趋势
- **板块温度**：按申万一级行业聚合的区间异动股数排行
- **区间异动榜**：正在进行中的上涨趋势 Top 20
- **单日异动榜**：当日脉冲式异动 Top 20

## 更新

每个交易日 18:00 / 20:00 / 22:00 自动更新（雪球数据更新后）。
